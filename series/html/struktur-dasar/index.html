Struktur Dasar HTML

Fundamental untuk mulai belajar pemrograman berbasis web

HTML (Hypertext Markup Language) adalah bahasa pemrograman yang digunakan untuk membuat halaman web. Struktur dasarnya seperti contoh code berikut

<html> <!-- awal dokumen html -->
  <head> <!-- element pendukung dokumen disimpan dalam head -->
    <title>Judul Laman</title> <!-- judul halaman -->
  </head>
  <body>
    <!-- Konten untuk ditampilkan ditulis disini (dalam body) -->
    Hello World!
  </body>
</html>

Kode bahasa pemrograman HTML ditulis dengan teks editor. Sebagai permulaan silakan salin kode di atas, masukkan dalam dokumen teks, dan simpan dalam format html (nama file: index.html)

Perhatikan secara seksama, disetiap <sintaks> selalu diikuti (ditutup) dengan </sintaks>. Hal ini dimaksudkan untuk menandai barisan kode yang ditulis merupakan bagian dari sebuah blok. Perhatikan sintaks title yang berada dalam blok head, menandakan bagian head bisa memprosesnya sebagai sebuah bagian dari dirinya (head), begitu sebaliknya bagian body tidak bisa mengakses kode title tersebut.

Biasakan untuk menuliskan <sintaks> sepasang sekaligus dengan penutupnya </sintaks> yang ditandai dengan backslash didepan sintaks yang kedua
